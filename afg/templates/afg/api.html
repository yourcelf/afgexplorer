Diary Dig API

The Diary Dig API exposes the basic functionality of Diary Dig to developers. To use, append "json" to the same URL used to display entries or search results, for example:

    http://wardiary.wikileaks.org/search/json?q=Obama...
    http://wardiary.wikileaks.org/id/2F8FCFDD-F5CB-9FAB-B96C7574D4BE7364/json

Results include all the basic information used to render pages in DiaryDig. For entry pages, this includes all the information available for that entry, as well as all the phrases that are linked to other documents, and report_key's that reference those documents. For search pages, this includes search results, facets, and parameters.

JSONp can be obtained by appending a "callback=..." GET parameter.

If you have questions, comments, or issues with the API, please use the github project page.
