Diary Dig API

The Diary Dig API exposes the basic functionality of Diary Dig to developers. To use, append "json" to the same URL used to display entries or search results, for example:

    http://www.diarydig.org/search/json?q=Obama...
    http://www.diarydig.org/id/2F8FCFDD-F5CB-9FAB-B96C7574D4BE7364/json

Results include all the basic information used to render pages in DiaryDig. For entry pages, this includes all the information available for that entry, as well as all the phrases that are linked to other documents, and ID's that reference those documents. For search pages, this includes search results, facets, and parameters.

JSONp can be obtained by appending a "callback=..." GET parameter.

Please note that currently search results are referenced by report_key, which is the unique ID provided by the WikiLeaks release data, while internal page links are referenced by an internal Diary Dig ID which may change. Both IDs can be used to access individual entries in Diary Dig.

If you have questions, comments, or issues with the API, please use the github project page.
